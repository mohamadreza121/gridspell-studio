# Project brief

Build GridSpell's public sales website, internal studio dashboard, and secure client portal as one premium platform.

## Objectives

1. Make quality immediately visible.
2. Explain the offer without technical confusion.
3. Attract buyers with realistic budgets.
4. Prove capability through case studies and process.
5. Convert visitors into structured project inquiries.
6. Give signed clients a premium delivery experience.
7. Give the owner one place to manage leads, projects, approvals, content, and billing.

## Primary conversion

Start a Project → structured brief → lead pipeline → discovery → proposal → deposit → client invitation.

## Non-goals for version one

- Full Figma replacement
- Custom accounting platform
- Real-time chat before basic messaging works
- White-label/multi-agency accounts
- Instant automated quotes
- Heavy 3D across every page
