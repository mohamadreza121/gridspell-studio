# MVP scope

## Public

Responsive navigation, homepage, work, three case studies, six service pages, process, pricing, about, inquiry, contact, legal placeholders, metadata, sitemap, robots, Open Graph, reduced-motion support.

## Admin

Lead submission/pipeline, notes/follow-up, organization/client/project creation, invitation, basic proposal, invoice visibility.

## Client

Login, project overview, phases, milestones, tasks/content requests, files, messages, approvals, billing status.

## Postponed

Live chat presence, pinned visual feedback, custom e-signature, proposal PDF generation, white label, resource scheduling, AI content, automated quotes, mobile app.

Done means a prospect can inquire, owner can create a project, client only sees assigned records, files/messages/approval work, billing status is visible, and authorization is enforced server-side and through RLS.
