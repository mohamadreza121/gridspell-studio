# Sitemap and user flows

## Public

`/`, `/work`, `/work/[slug]`, `/services`, `/services/[slug]`, `/process`, `/pricing`, `/about`, `/insights`, `/contact`, `/start-project`, `/login`, `/privacy`, `/terms`.

## Client

`/portal`, projects, milestones, tasks, approvals, files, messages, billing, support, settings.

## Admin

`/admin`, leads, clients, projects, proposals, invoices, content, settings.

## Prospect flow

Referral/search/social → hero → work → matching service → process/pricing → project brief → lead pipeline → discovery → proposal → deposit → account invitation.

## Client flow

Login → progress/current phase → outstanding requests → files/content → approvals/feedback → billing → launch/support.

## Owner flow

Qualify lead → discovery notes → proposal → convert to organization/client/project → invite → milestones/content/approvals → invoice → case study.
