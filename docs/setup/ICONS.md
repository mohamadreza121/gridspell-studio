# Browser and app icons

```text
src/app/
├── favicon.ico
├── icon.png
└── apple-icon.png

public/
├── android-chrome-192x192.png
├── android-chrome-512x512.png
└── site.webmanifest
```

Next.js recognizes the App Router icon files automatically. The manifest points to the Android files. Replace the placeholders without renaming them.
